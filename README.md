# 🤖 Week 1: Rule-Based AI Chatbot

## 📌 Project Overview

This project is a simple rule-based AI chatbot developed using Python. The chatbot responds to user inputs based on predefined rules and simulates basic human-like conversation.

---

## 🎯 Objective

* Understand basic AI concepts
* Build a chatbot using control flow and logic
* Learn how machines respond to user input

---

## ⚙️ Features

* Takes user input continuously
* Matches input using predefined rules
* Responds with appropriate answers
* Handles unknown inputs with a default reply
* Exit option to stop the chatbot

---

## 🛠️ Technologies Used

* Python
* Basic Control Flow (if-else / dictionary mapping)

---

## ▶️ How to Run

1. Open the Python file
2. Run the program
3. Enter your message in the console
4. Type `exit` to stop the chatbot

---

## 💡 Sample Interaction

You: hello
Bot: Hi! How can I help you today?

You: bye
Bot: Goodbye!

---

## 📚 Key Concepts Learned

* Rule-based systems
* Input handling
* Conditional logic
* Basic chatbot design

---

## 🏁 Conclusion

This project demonstrates how a simple chatbot can be built using logic and predefined responses. It serves as a foundation for more advanced AI systems.
